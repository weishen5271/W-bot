# 稀土掘金博客内容创作指南

## 平台定位

稀土掘金是面向开发者的技术社区，内容以技术分享为主，读者群体主要是程序员、技术人员。

## 内容方向

### 受欢迎的内容类型

1. **技术教程**
   - 新技术的入门到实战
   - 框架/库的使用详解
   - 最佳实践总结

2. **源码解析**
   - 开源项目源码分析
   - 框架设计思想
   - 实现原理探讨

3. **实战经验**
   - 项目踩坑记录
   - 性能优化案例
   - 架构设计分享

4. **技术趋势**
   - 新技术评测
   - 行业动态分析
   - 学习路线推荐

### 避免的内容

- 纯广告软文
- 与技术无关的生活随笔
- 过于基础无技术含量的内容
- 大量复制粘贴的转载内容

## 文章结构建议

### 标题

- 简洁明确，包含关键词
- 可以使用数字、问句等形式
- 示例：
  - "React Hooks 深入浅出：从原理到实践"
  - "10个JavaScript性能优化技巧"
  - "为什么你的Redis变慢了？"

### 开头（引言）

- 说明文章要解决的问题
- 简述文章价值
- 可以引用数据或案例

### 正文

- 使用多级标题组织内容
- 代码示例要完整可运行
- 配图说明（架构图、流程图等）
- 适当使用列表、表格

### 结尾（总结）

- 回顾核心要点
- 给出行动建议
- 可以引导讨论或关注

## 代码规范

### 代码块

- 指定语言以获得语法高亮
- 包含必要的注释
- 示例要完整可运行

```python
# 良好的示例

def fibonacci(n: int) -> int:
    """计算斐波那契数列第n项"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# 测试
print(fibonacci(10))  # 55
```

### 避免的问题

- 代码不完整，无法运行
- 缺少必要的导入或配置
- 没有输出示例

## SEO 优化建议

1. **关键词**: 标题和正文中自然融入关键词
2. **摘要**: 写出吸引人的文章摘要
3. **标签**: 选择相关的技术标签
4. **封面图**: 添加吸引人的封面图片
5. **外链**: 适当引用权威来源

## 发布时机

- 工作日早晚高峰（8-9点，20-22点）
- 避开节假日和周末（技术社区活跃度较低）
- 保持稳定的更新频率
